# Automated-DCA-Pipeline
Automated Decline Curve Analysis Pipeline berbasis Python untuk menganalisis data produksi minyak secara otomatis melalui data cleaning, deteksi shut-in/workover, segmentasi riwayat produksi, pemilihan segmen terbaik, fitting model Arps (Exponential, Harmonic, Hyperbolic), forecasting produksi, dan estimasi EUR guna mempercepat analisis reservoir.
